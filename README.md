# JapaneseKana - 日语五十音学习应用

这是一个基于 Android Jetpack Compose 开发的简单易用的日语五十音学习应用。它可以帮助初学者快速掌握平假名、片假名及其发音。

## 功能特点

*   **完整图表**：包含完整的五十音图（清音）、浊音/半浊音以及拗音。
*   **灵活切换**：支持平假名优先或片假名优先的显示模式，方便对比学习。
*   **测验模式**：内置测试功能，通过随机题目检验学习成果，巩固记忆。
*   **现代 UI**：采用 Material 3 设计语言，界面简洁直观，支持响应式布局。

## 技术栈

*   **编程语言**：Kotlin
*   **界面框架**：Jetpack Compose
*   **架构组件**：ViewModel, LiveData/State
*   **核心库**：
    *   `androidx.compose.material3`: 提供现代化的 UI 组件。

## 安装与运行

1.  克隆或下载本项目。
2.  使用 Android Studio (推荐 Ladybug 或更新版本) 打开项目。
3.  确保已配置好 Android SDK。
4.  连接 Android 设备或启动模拟器（建议 API 26 及以上）。
5.  点击 "Run" 按钮即可安装并运行。
